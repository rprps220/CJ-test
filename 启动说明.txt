AI对话助手 - 启动说明
========================

由于浏览器安全限制，直接打开HTML文件可能无法正常调用API。
请按照以下步骤启动：

方法1：使用Python服务器（推荐）
------------------------------
1. 打开命令提示符或PowerShell
2. 进入项目文件夹：cd C:\Users\w550s\Desktop\9.14
3. 启动服务器：python -m http.server 8000
4. 打开浏览器访问：http://localhost:8000

方法2：使用批处理文件
--------------------
1. 双击运行 simple-server.bat 文件
2. 等待服务器启动
3. 打开浏览器访问：http://localhost:8000

方法3：使用Node.js（如果已安装）
--------------------------------
1. 打开命令提示符
2. 进入项目文件夹
3. 运行：npx serve . -p 8000
4. 打开浏览器访问显示的地址

故障排除
--------
如果遇到问题：
1. 确保Python已安装
2. 检查端口8000是否被占用
3. 尝试使用其他端口（如8001）
4. 检查防火墙设置

API配置
-------
1. 打开 config.js 文件
2. 在 DEEPSEEK_API_KEY 字段填入你的API Key
3. 保存文件并刷新浏览器

联系支持
--------
如果仍有问题，请检查：
- 网络连接
- API Key是否正确
- 浏览器控制台错误信息
